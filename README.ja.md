# Scene Tab Manager for Godot 4.x

[English](README.md)

**Scene Tab Manager** は、Godot エディタで開いているたくさんのシーンタブを、キーワードの優先度（重み）に合わせて自動で並べ替えてくれるプラグイン。

大きなプロジェクトで作業しているとメインメニューやプレイヤー、共通のベースシーンといった「よく使うシーン」が他のタブに埋もれてしまいがちである。
そこでこのプラグインを使えばそんな悩みもスッキリ解決。

## 主な機能

* **キーワードで並べ替え**: シーンのファイル名に含まれるキーワード（デフォルト: `title`, `level_base`, `player`）を見て、優先度の高い順に並べ替える。
* **カスタマイズも簡単**: **エディタ設定**、またはボタンの右クリックメニューからキーワードのスコアやショートカットの設定を変更できる。
* **ワンクリック操作**: 2D/3D ビューポートのツールバーに追加される「整理（Organize）」ボタンを押すだけで整理が完了する。
* **最近使ったファイル (Ctrl+E)**: 最近開いたシーンやスクリプトを、検索可能なポップアップから素早く切り替える。
* **クイックオープン (Shift 2回押し)**: **Shift** キーを素早く2回押すことで、リソース検索ダイアログを起動する。
* **クイックファイル特定 (Alt+クリック)**: **Alt** キーを押しながらノードやリソースをクリックすると、そのファイルを **FileSystem** ドックで即座に表示する。
* **タブ切り替え (Alt + 1-9)**: 数字キーを使って、最初の9つのシーンタブを一瞬で切り替える。
* **シグナル接続のツールチップ表示**: **シーンツリー**のノードをホバーすると、そのノードのシグナル接続状況を一覧表示する。
* **自動クリーンアップ**: ファイルが削除された際、開いている無効なタブを自動的に閉じる。

## インストール

1.  `addons/scene_tab_manager` フォルダを、自分のプロジェクトの `addons/` の中にコピーする。
2.  Godot エディタの **プロジェクト -> プロジェクト設定** を開く。
3.  **プラグイン** タブで **Scene Tab Manager** を **有効** にする。

## 使い方

### タブを整理する
エディタ上部のツールバーにある **Organize** ボタン（フォルダのアイコン）をクリック。

![Organize Button Location](doc_images/info_icon0.jpg)

### 設定へのクイックアクセス (右クリック)
**Organize** ボタンを右クリックして素早く操作：
* **Organize Tabs**: 左クリックと同じ。
* **Edit Keyword Weights...**: インスペクターでプラグインの設定項目（重みとショートカット設定）を開く。
* **Reset to Default**: 設定を初期状態に戻す。

### 最近使ったファイル (Ctrl+E)
**Ctrl+E** を押すと、最近使ったファイルのポップアップが表示される。
* **矢印キー**で選択。
* 数字キー **1-9**（および **0**）で、対応するインデックスのファイルをフォーカスし、**Enter**キーで開く。

### クイックオープン (Shift 2回押し)
**Shift** キーを2回連続で押すと、リソース検索ダイアログが開く。
* **Alt + 決定**: ファイルを開く代わりに、FileSystem ドックでその場所を表示する。

### タブ切り替え (Alt + 1-9)
**Alt** を押しながら数字キー (**1-9**) を押すと、対応するタブへジャンプする。この際、エディタは自動的に **2D ビュー** に切り替わる。

### クイックファイル特定 (Alt+クリック)
**Alt** キーを押しながら以下をクリックすると、FileSystem ドックでそのファイルが強調表示される。
* **シーンツリーのノード**: 保存されたシーン（インスタンス）なら、そのシーンファイルを表示。
* **インスペクターのリソース**: テクスチャやスクリプトなどのプロパティをクリック。

### シグナル接続のツールチップ表示
**シーンツリー**ドックのノードにマウスを合わせると、接続情報が表示される。
* **Signals (Outgoing)**: このノードからの接続。
* **Incoming**: このノードへの接続。
* **詳細表示 (Ctrlキーを押しながらホバー)**: エンジン内部の接続を含むすべての情報を表示。

### 設定項目
全ての設定項目は、**エディタ -> エディタ設定 -> Editors -> Plugins -> Scene Tab Manager** から変更可能です：
* `keyword_weights`: キーワードと重み（スコア）の辞書（デフォルト: `{"title": 50, "level_base": 30, "player": 10}`）。
* `shortcuts/recent_files`: 「最近使ったファイル」のポップアップを開くショートカットキー（デフォルト: `Ctrl+E`）。
* `shortcuts/enable_double_shift`: **Shift** キーの2回押しによる「クイックオープン」ダイアログ起動の有効化（デフォルト: `true`）。
* `shortcuts/enable_alt_tab_switching`: **Alt + 1-9** によるタブ切り替えの有効化（デフォルト: `true`）。
* `shortcuts/enable_alt_click_locate`: **Alt + クリック** による FileSystem ドックでのファイル特定機能の有効化（デフォルト: `true`）。

## 技術的な話と制限

* **フレーム単位のソート**: タブの並べ替えはエディタの安定性を保つため、複数フレームに分けて実行される。
* **フォーカス復元**: 並べ替えが終わると、自動的に元のシーンにフォーカスを戻す。
* **最適化**: ツールチップの更新間隔は0.1秒。誤作動防止のため、ファイル特定アクションには0.25秒のクールダウンを設けている。
* **自動クリーンアップ**: ファイルシステムの変更を監視し、ファイルが移動・削除されたタブを自動的に閉じる。
